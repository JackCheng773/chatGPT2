### 15.4 用 New Bing 取得網頁摘要

**SETP 1**：進入你想要摘要網址，ex：https://medium.com/dean-lin/b7bba6a1f01f

**SETP 2**：在右側的「聊天」輸入「幫我現在瀏覽的網頁做摘要」即可。

> 要讓他摘要英文網站，可以輸入「幫我現在瀏覽的網頁做摘要，並用繁體中文呈現」即可。
